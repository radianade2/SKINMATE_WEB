# FINAL PROJECT WEBPRO
SKINMATE: WEATHER SKINCARE RECOMMENDATION

### WEBSITE DESCRIPTION
This dummy website uses Laravel framework. <br> 
The first page displays homepage, which contains information about skincare.<br> 
The second page contains a form to collect user input. <br> 
The third page contains product recommendations for the user based on the user's answers on the form. <br>

Some of the decision-making parameters are as follows: <br>
- Location (Used to get temperature (in Celsius) and UV Index from API)
- Gender
- Age
- Skintype
- Sensitivity
- Activity (indoor/outdoor)
- Sun allergy
- Pregnancy

Location data retrieval is required to GET temperature and UV index data from that location using the API at https://api.openweathermap.org/data/2.5/weather.
